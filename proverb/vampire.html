Tool

PV0 ‚äß Vampire

First-order theorem prover

Application domain/field

  • Theorem proving
  • First-order logic

Type of tool

Automatic theorem prover

Expected input

  • First-order logic theorem
  • Optional: Memory-limit and/or time-limit

Format:

  • First-order logic theorem: TPTP syntax (.tptp file)
  • Memory/time-limit: parameter when calling Vampire

Expected output

Reason for termination (refutation was found, time/memory limit was reached, satisfiability detected, ...) It also prints some statistics about the proof attempt.

Internals

Uses MiniSat, Z3.
