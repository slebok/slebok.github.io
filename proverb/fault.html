PV2 ⊧ faultgenerates tests for a circuit

Application domain/field

  • Hardware verification
  • Computer-aided design

Type of tool

Test generator framework

Expected input

The user should construct a Tester object with a magma circuit. The user should then record a sequence of test actions using the Tester's API class.

Format:

Python code

Expected output

Whether the tests pass or fail

Internals

fault is an embedded domain-specific language that allows the user to construct test generators. It has been designed to support magma, which is an embedded hardware construction language. It has several different backend targets including Verilog, SPICE, Verilog-AMS for simulation and CoSA for model checking.
