Tool

SAT

PV4 ⊧ plingelingproduces a satisfiability result for a formula

Parallel front-end of Lingeling SAT solver

Application domain/field

  • SAT solving
  • Boolean satisfiability

Type of tool

SAT solver

Expected input

SAT problem

Format:

DIMACS

Expected output

SATISFIABLE, UNSATISFIABLE or UNKNOWN indicating whether the SAT problem is satisfiable, unsatisfiable or it could not be determined.
