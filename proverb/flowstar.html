Tool

PV0 ⊧ Flow*

Application domain/field

  • Hybrid systems
  • Non-linear hybrid systems
  • Safety verification
  • Reachability analysis

Expected input

  • Hybrid system model file which describes the modes, the polynomial dynamics associated with each mode and the transitions between modes
  • A specification file includes Taylor Model flowpipes with the state space and unsage set specifications

Expected output

Flow* computes an over-approximation of the reachable states and checks whether this intersects with the unsafe set. It will output a visualization of the set of reachable states using polyhedral over-approximations of the computed Taylor Model flowpipes.
