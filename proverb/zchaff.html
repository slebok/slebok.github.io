Tool

SAT

PV4 ⊧ zChaff (sometimes stylized as zCHAFF)produces a satisfiability result for a formula in CNF

SAT solver

Application domain/field

  • SAT solving
  • Boolean satisfiability

Type of tool

SAT solver

Expected input

SAT problem

Format:

CNF file

Expected output

SAT, UNSAT, ABORT : TIME OUT, ABORT : MEM OUT, UNKNOWN indicating whether the SAT problem is satisfiable, unsatisfiable, or it could not be determined.

Internals

zChaff is an implementation of the Chaff algorithm.
