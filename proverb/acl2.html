PV6 ⊧ ACL2: A Computational Logic for Applicative Common Lispa full-fledged language to write specs and prove their properties

Logic and programming language which you can use to model computer systems, and a tool to help you prove properties of those models. i.e. theorem prover

Application domain/field

  • Theorem prover
  • Rewriting
  • Interactive theorem prover

Type of tool

Framework/Theorem prover
