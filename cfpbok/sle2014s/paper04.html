SLE 2014 SI paper:
Intra- and interdiagram consistency checking of behavioral multiview models

Petra Radeljak Kaufmann, Martin Kronegger, Andreas Pfandler, Martina Seidl, Magdalena Widl

  • DOI: 10.1016/j.cl.2015.08.003
  • Extended version of A SAT-Based Debugging Tool for State Machines and Sequence Diagrams (SLE 2014)
  • T5D: Formal Methods
    • The contribution is a formal consistency-checking method for behavioral models (with SAT solving as the underlying decision procedure).
  • T5B: Language Description Analysis
    • It analyses language artifacts (behavioral UML models/diagrams) to detect and explain inconsistencies across views.
  • T3C: DSLs
    • The work targets a domain/purpose-specific modeling setting (behavioral multiview models) rather than general-purpose programming.
