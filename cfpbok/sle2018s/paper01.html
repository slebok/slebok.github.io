SLE 2018 SI paper:
Relational Reference Attribute Grammars: Improving continuous model validation

Johannes Mey, René Schöne, Görel Hedin, Emma Söderberg, Thomas Kühn, Niklas Fors, Jesper Öqvist, Uwe Aßmann

  • DOI: 10.1016/j.cola.2019.100940
  • Extended version of Continuous model validation using reference attribute grammars (SLE 2018)
  • T3A: Meta-languages
    • The paper extends Reference Attribute Grammars with first-class relations (“Relational RAGs”), advancing a meta-language framework for defining incremental analyses over models/ASTs.
  • T4A: Workbenches
    • It is realised in a RAG toolchain (JastAdd) with engineering for demand-driven, incremental evaluation plus persistence/serialization support.
  • T2B: Evolution
    • Its central scenario is continuous validation under ongoing model edits, addressing maintenance-like concerns through incremental recomputation strategies.
  • T5E: Empirical Evaluation
    • It reports empirical results (e.g., on benchmarked models/strategies) to quantify performance and responsiveness trade-offs for incremental validation.
