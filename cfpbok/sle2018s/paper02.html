SLE 2018 SI paper:
MORBIG: A Static Parser for POSIX Shell

Yann Régis-Gianas, Nicolas Jeannerod, Ralf Treinen

  • DOI: 10.1016/j.cola.2020.100944
  • Extended version of Morbig: a static parser for POSIX shell (SLE 2018)
  • T4C: Vertical Transformation
    • The main contribution is a static parser for POSIX shell that resolves shell-specific syntactic challenges, i.e., a compilation-style front-end component.
  • T3D: GPLs
    • The work targets the POSIX shell language, a widely used general-purpose scripting/programming language.
  • T5E: Empirical Evaluation
    • The paper validates correctness/practicality via substantial evaluation on real-world scripts and comparisons against existing implementations.
