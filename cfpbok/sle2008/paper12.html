SLE 2008 paper:
A Practical Evaluation of Using TXL for Model Transformation

Hongzhi Liang, Jürgen Dingel

  • DOI: 10.1007/978-3-642-00434-6_16
  • T5E: Empirical Evaluation
    • It is explicitly an evaluation of using TXL for model transformation, focusing on practical evidence and assessment.
  • T4B: Horizontal Transformation
    • TXL is used for performing transformations, so model-to-model / source-to-source transformation is central.
  • T3B: Transformation Languages
    • The work assesses a specific transformation-language technology (TXL) in the model-transformation setting.
