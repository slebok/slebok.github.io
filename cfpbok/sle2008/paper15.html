DeFacto: Language-Parametric Fact Extraction from Source Code

Hendrikus J. S. Basten, Paul Klint

  • DOI: 10.1007/978-3-642-00434-6_17
  • T5B: Language Description Analysis
    • It provides language-parametric fact extraction that underpins systematic language/source-code analysis.
  • T4A: Workbenches
    • The contribution includes a reusable extraction/analysis infrastructure intended to be plugged into tools.
  • T1E: Reuse
    • Language-parametricity and separable annotations enable reuse of the same extractor across many languages.
