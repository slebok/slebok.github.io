SLE 2008 paper:
The Java Programmer’s Phrase Book

Einar W. Høst, Bjarte M. Østvold

  • DOI: 10.1007/978-3-642-00434-6_20
  • T5E: Empirical Evaluation
    • It builds an automatically generated phrase book from a large corpus of Java programs, relying on empirical analysis of real-world usage.
  • T5B: Language Description Analysis
    • It analyzes the structure and ‘semantics’ of method-name phrases, treating identifier naming as a describable language.
  • T2B: Evolution
    • The motivation includes supporting developers as code and naming conventions evolve (e.g., guiding/refactoring names).
