SLE 2024 SI paper:
A language-parametric test coverage framework for executable domain-specific languages

Faezeh Khorram, Erwan Bousse, Antonio Garmendía, Jean-Marie Mottu, Gerson Sunyé, Manuel Wimmer

  • DOI: 10.1016/j.jss.2024.111977
  • T5C: Testing
    • The main contribution is a generic test coverage framework/metrics (model element & branch coverage) for testing executable DSL models.
  • T3C: DSLs
    • The approach is designed specifically for executable domain-specific languages (xDSLs) and their models/traces.
  • T4A: Workbenches
    • It is implemented in/for a language & modeling workbench setting (Eclipse GEMOC Studio) with a dedicated coverage rule language.
  • T5E: Empirical Evaluation
    • The paper evaluates the framework across multiple xDSLs, models, test suites, and fault-localization scenarios.
