SLE 2024 SI paper:
Grammar-based test suite construction using coverage-directed algorithms over LR-graphs

Christoff Rossouw, Bernd Fischer

  • DOI: 10.1016/j.jss.2024.112068
  • Extended version of Test Case Generation from Context-Free Grammars using Generalized Traversal of LR-automata (SLE 2020)
  • T5C: Testing
    • The contribution is an algorithm for generating grammar-based test suites that satisfy coverage criteria via LR-automaton graph exploration.
  • T5B: Language Description Analysis
    • It relies on analysis of grammar-derived LR automata/graphs (push/pop edges, conflicts) to guide construction of valid derivations.
  • T5E: Empirical Evaluation
    • The paper empirically evaluates efficiency/effectiveness (including comparisons to prior LR-graph exploration approaches).
