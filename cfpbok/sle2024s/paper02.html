SLE 2024 SI paper:
Supporting reusable model migration with Edelta

Lorenzo Bettini, Amleto Di Salle, Ludovico Iovino, Alfonso Pierantonio

  • DOI: 10.1016/j.jss.2024.112012
  • T2B: Evolution
    • The work targets model/metamodel evolution, specifically supporting model migration across evolving metamodels.
  • T3B: Transformation Languages
    • Edelta is presented as a dedicated language for specifying migrations/operations (i.e., transformation specifications) in a reusable way.
  • T4A: Workbenches
    • The contribution includes tooling/framework support to automate and apply migrations in a language engineering ecosystem.
  • T1E: Reuse
    • Emphasis is on reusable migration building blocks/operators that can be applied across multiple evolution scenarios.
