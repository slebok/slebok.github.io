SLE 2011 paper:
A Dedicated Language for Context Composition and Execution of True Black-Box Model Transformations

Andreas Seibel, Regina Hebig, Stefan Neumann, Holger Giese

  • DOI: 10.1007/978-3-642-28830-2_2
  • T1D: Composition
    • The work introduces a composition framework for coordinating heterogeneous model transformations as black boxes.
  • T3B: Transformation Languages
    • It targets composition across different model transformation languages and engines.
  • T5G: Traceability
    • Traceability is explicitly part of the problem setting and keywords, to relate composed transformations and results.
  • T6B: Industrial
    • The approach is illustrated on an industrial case study.
