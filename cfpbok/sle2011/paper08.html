SLE 2011 paper:
An Architecture for Information Exchange Based on Reference Models

Heiko Paulheim, Daniel Oberle, Roland Plendl, Florian Probst

  • DOI: 10.1007/978-3-642-28830-2_9
  • T1D: Composition
    • The core contribution is an integration architecture enabling information exchange across heterogeneous IT systems.
  • T4B: Horizontal Transformation
    • It relies on flexible bidirectional mappings between implementation models and a reference model.
  • T3E: Ontologies
    • The reference model is an ontology (ISO 15926 in OWL/RDF), making ontological modeling central to the approach.
  • T6B: Industrial
    • The feasibility is demonstrated in an industrial case study (SAP Research / Oil & Gas domain).
