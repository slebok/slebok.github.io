SLE 2011 paper:
RLSRunner: Linking Rascal with K for Program Analysis

Mark Hills, Paul Klint, Jurgen Vinju

  • DOI: 10.1007/978-3-642-28830-2_19
  • T4A: Workbenches
    • RLSRunner is a tool that integrates Rascal with K-based analyzers, functioning as a meta-tool/workbench component.
  • T3A: Meta-languages
    • Both Rascal and K are meta-languages used to specify analyses and semantics.
  • T5B: Language Description Analysis
    • The purpose of the integration is program analysis driven by language definitions and semantics.
