SLE 2011 paper:
MoScript: A DSL for Querying and Manipulating Model Repositories

Wolfgang Kling, Frédéric Jouault, Dennis Wagelaar, Marco Brambilla, Jordi Cabot

  • DOI: 10.1007/978-3-642-28830-2_10
  • T3C: DSLs
    • MoScript is introduced as a DSL tailored to querying and manipulating model repositories.
  • T3F: API
    • It provides an interface layer for accessing repository functionality via scripting queries and updates.
  • T4B: Horizontal Transformation
    • Repository queries and updates operationalise transformations over stored models and their representations.
