SLE 2014 paper:
Test-Data Generation for Xtext (Tool Paper)

Johannes Härtel, Lukas Härtel, Ralf Lämmel

  • DOI: 10.1007/978-3-319-11245-9_19
  • T5C: Testing
    • The contribution is a method and tool for grammar-based test-data generation for Xtext languages.
  • T4A: Workbenches
    • It targets tooling support around Xtext language development and testing.
  • T3B: Transformation Languages
    • It uses a DSL for grammar transformation to customize grammars for test-data generation.
  • T5B: Language Description Analysis
    • It relies on grammatical structure (parse trees/choices) and language descriptions to drive generation.
