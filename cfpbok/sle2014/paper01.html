SLE 2014 paper:
ProMoBox: A Framework for Generating Domain-Specific Property Languages

Bart Meyers, Romuald Deshayes, Levi Lúcio, Eugene Syriani, Hans Vangheluwe, Manuel Wimmer

  • DOI: 10.1007/978-3-319-11245-9_1
  • T5D: Formal Methods
    • The paper centers on model checking of domain models via domain-specific property specification and verification.
  • T3C: DSLs
    • It generates domain-specific property languages so domain experts can state temporal properties in familiar terms.
  • T4A: Workbenches
    • It provides a framework that integrates property modeling and result visualisation into DSM environments.
  • T4B: Horizontal Transformation
    • It relies on automated translation to verification back-ends (e.g., mappings to existing model checkers).
