SLE 2014 paper:
Dynamic Scope Discovery for Model Transformations

Māris Jukšs, Clark Verbrugge, Dániel Varró, Hans Vangheluwe

  • DOI: 10.1007/978-3-319-11245-9_17
  • T5F: Performance
    • Its main aim is runtime performance optimisation of graph pattern matching in model transformations.
  • T3B: Transformation Languages
    • The technique is embedded in a model transformation engine and targets transformation execution.
  • T4D: Interpretation
    • It operates at runtime by dynamically reducing the search space for rule application.
  • T4G: AI-for-SLE
    • It employs a Naive Bayes classifier and online learning to predict likely match candidates.
