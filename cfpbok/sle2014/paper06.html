SLE 2014 paper:
The Moldable Debugger: A Framework for Developing Domain-Specific Debuggers

Andrei Chiș, Tudor Gîrba, Oscar Nierstrasz

  • DOI: 10.1007/978-3-319-11245-9_6
  • Original shorter version of Practical domain-specific debuggers using the Moldable Debugger framework (SLE 2014 SI)
  • T4A: Workbenches
    • It proposes a debugger framework intended as a reusable workbench for building domain-specific debuggers.
  • T4D: Interpretation
    • It targets debugging of running systems, i.e., runtime inspection and interaction.
  • T3C: DSLs
    • It supports domain-specific debugging concepts/views tailored to particular domains/languages.
  • T5H: Usability
    • The motivation is to improve the effectiveness of debugging by offering domain-appropriate interactions.
