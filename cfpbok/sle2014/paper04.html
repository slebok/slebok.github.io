SLE 2014 paper:
Bounded Seas: Island Parsing Without Shipwrecks

Jan Kurš, Mircea Lungu, Oscar Nierstrasz

  • DOI: 10.1007/978-3-319-11245-9_4
  • Original shorter version of Bounded seas (SLE 2014 SI)
  • T5B: Language Description Analysis
    • The core contribution is a parsing technique (island grammars) for robust/partial language processing.
  • T2A: Lifecycle
    • It supports error recovery and robust parsing when dealing with incomplete or noisy source input.
  • T3D: GPLs
    • The technique is motivated by practical manipulation of general-purpose programming language code.
