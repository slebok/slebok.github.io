SLE 2014 paper:
Origin Tracking in Attribute Grammars

Kevin Williams, Eric Van Wyk

  • DOI: 10.1007/978-3-319-11245-9_16
  • T3A: Meta-languages
    • It extends higher-order attribute grammars with origin tracking as part of the metalanguage machinery.
  • T5G: Traceability
    • Origin tracking provides traceability links from transformation outputs back to their inputs.
  • T3B: Transformation Languages
    • It relates to tree transformation/rewriting by tracking how constructed trees correspond to source trees.
  • T1C: Behavioural Semantics
    • It concerns the semantics of attribute evaluation that constructs new syntax trees.
