SLE 2014 paper:
Streamlining Control Flow Graph Construction with DCFlow

Mark Hills

  • DOI: 10.1007/978-3-319-11245-9_18
  • T3C: DSLs
    • It introduces DCFlow, a DSL for specifying control-flow rules at the language AST level.
  • T4C: Vertical Transformation
    • DCFlow definitions are compiled into generated Rascal code that constructs CFGs.
  • T4A: Workbenches
    • It is delivered as a DSL plus libraries supporting CFG extraction within a workbench setting.
  • T4F: Visualisation
    • Generated CFGs can be visualized, and the paper discusses visualization as a use of the produced graphs.
