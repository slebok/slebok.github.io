SLE 2012 paper:
The Program Is the Model: Enabling Transformations@run.time

Jesús Sánchez Cuadrado, Esther Guerra, Juan de Lara

  • DOI: 10.1007/978-3-642-36089-3_7
  • T4D: Interpretation
    • The focus is on enabling model-like transformations at runtime, by reflecting programs/APIs into a model representation.
  • T3F: API
    • APIs are treated as the primary “model” abstraction, enabling transformations over program structures exposed via APIs.
  • T3B: Transformation Languages
    • The paper operationalizes runtime change via transformation rules that modify program structures through the model view.
