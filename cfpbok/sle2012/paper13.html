SLE 2012 paper:
Layout-Sensitive Generalized Parsing

Sebastian Erdweg, Tillmann Rendel, Christian Kästner, Klaus Ostermann

  • DOI: 10.1007/978-3-642-36089-3_14
  • T4A: Workbenches
    • The paper’s main result is a generalized parsing technique/tooling that handles layout-sensitive syntax systematically.
  • T1D: Composition
    • Layout sensitivity is treated in a way that supports combining grammars and languages without losing compositionality.
  • T3A: Meta-languages
    • The work extends grammar-based specification techniques for defining syntaxes that include layout constraints.
