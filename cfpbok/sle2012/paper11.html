SLE 2012 paper:
Grammatical Inference in Software Engineering: An Overview of the State of the Art

Andrew W. Stevenson, James R. Cordy

  • DOI: 10.1007/978-3-642-36089-3_12
  • Original shorter version of A survey of grammatical inference in software engineering (SLE 2012 SI)
  • T4G: AI-for-SLE
    • The paper surveys grammar inference (learning grammars from data), which is an AI/ML-style technique applied to SLE tasks.
  • T3A: Meta-languages
    • The inferred artifacts are grammars/language definitions, i.e., meta-language objects central to software language engineering.
  • T5B: Language Description Analysis
    • It is concerned with recovering/analyzing language descriptions (grammars) from programs, traces, or other artifacts.
