SLE 2012 paper:
Object Grammars: Compositional and Bidirectional Mapping between Text and Graphs

Tijs van der Storm, William R. Cook, Alex Loh

  • DOI: 10.1007/978-3-642-36089-3_2
  • Original shorter version of The design and implementation of Object Grammars (SLE 2012 SI)
  • T3A: Meta-languages
    • It introduces “object grammars” as a meta-language to define syntax together with mappings between text and object graphs.
  • T1D: Composition
    • A central concern is compositionality—how grammars/mappings can be combined and embedded modularly.
  • T4D: Interpretation
    • The contribution supports bidirectional parsing/pretty-printing at runtime, i.e., executable mappings between representations.
