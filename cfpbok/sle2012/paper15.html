SLE 2012 paper:
TouchRAM: A Multitouch-Enabled Tool for Aspect-Oriented Software Design

Wisam Al Abed, Valentin Bonnet, Matthias Schöttle, Engin Yıldırım, Omar Alam, Jörg Kienzle

  • DOI: 10.1007/978-3-642-36089-3_16
  • T4F: Visualisation
    • The contribution is an interactive, multitouch visualization and manipulation environment for aspect-oriented design models.
  • T5H: Usability
    • Multitouch interaction is motivated by improving human usability of complex design operations and model manipulation.
  • T4A: Workbenches
    • It is a concrete tool implementation that supports editing/analysis workflows for aspect-oriented modeling.
