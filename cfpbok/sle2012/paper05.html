SLE 2012 paper:
Temporal Constraint Support for OCL

Bilal Kanso, Safouan Taha

  • DOI: 10.1007/978-3-642-36089-3_6
  • Original shorter version of Specification of temporal properties with OCL (SLE 2012 SI)
  • T5D: Formal Methods
    • It provides a formal, scenario-based semantics for temporal constraints on OCL and links this to formal-methods practice.
  • T5C: Testing
    • Temporal constraints are explicitly connected to automatic test generation via test purposes derived from the constraints.
  • T4A: Workbenches
    • A key practical contribution is an implementation integrated into the Eclipse/MDT OCL tooling.
