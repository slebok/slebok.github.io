SLE 2012 paper:
On the Reusable Specification of Non-functional Properties in DSLs

Francisco Durán, Steffen Zschaler, Javier Troya

  • DOI: 10.1007/978-3-642-36089-3_19
  • T1E: Reuse
    • The paper’s main goal is reusable DSL building blocks for specifying/analyzing non-functional properties across DSLs.
  • T3C: DSLs
    • The setting is the construction and extension of DSLs, including modular language components for NFPs.
  • T5D: Formal Methods
    • It provides formal conditions/proofs for conservative extension and supports analyses via model checking.
