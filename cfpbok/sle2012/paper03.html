SLE 2012 paper:
Termination Analysis for Higher-Order Attribute Grammars

Lijesh Krishnan, Eric Van Wyk

  • DOI: 10.1007/978-3-642-36089-3_4
  • Original shorter version of Monolithic and modular termination analyses for higher-order attribute grammars (SLE 2012 SI)
  • T5D: Formal Methods
    • The main contribution is a formal termination analysis technique for higher-order attribute grammars.
  • T5B: Language Description Analysis
    • It targets a meta-level correctness property (termination) of language definitions expressed as attribute grammars.
  • T3A: Meta-languages
    • Higher-order attribute grammars are treated as a meta-language whose specifications are analyzed.
