SLE 2012 paper:
Approaches and Tools for Implementing Type Systems in Xtext

Lorenzo Bettini, Dietmar Stoll, Markus Völter, Serano Colameo

  • DOI: 10.1007/978-3-642-36089-3_22
  • T1B: Static Semantics
    • The paper is about defining and implementing type systems (type calculation/conformance) for Xtext-based DSLs.
  • T4A: Workbenches
    • It compares approaches within and around the Xtext language workbench, focusing on tool support and integration.
  • T5H: Usability
    • Usability (required effort, flexibility, and developer experience) is an explicit comparison criterion for the surveyed approaches.
