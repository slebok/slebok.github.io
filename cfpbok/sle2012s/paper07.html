SLE 2012 SI paper:
Specification of temporal properties with OCL

Bilal Kanso, Safouan Taha

  • DOI: 10.1016/j.scico.2014.02.029
  • Extended version of Temporal Constraint Support for OCL (SLE 2012)
  • T5D: Formal Methods
    • The main contribution is a formal property-specification framework (temporal extension of OCL) with formal semantics and translation to automata.
  • T1C: Behavioural Semantics
    • The paper adds behavioural/temporal semantics (scenario-based) on top of OCL’s traditionally static constraint focus.
  • T4A: Workbenches
    • Tooling is central: the approach is integrated into the Eclipse OCL plug-in and supported by an implementation pipeline.
  • T5C: Testing
    • The framework is connected to automatic test generators via the automata translation, tying the contribution to testing practice.
