SLE 2010 paper:
Reference Attribute Grammars for Metamodel Semantics

Christoff Bürger, Sven Karol, Christian Wende, Uwe Aßmann

  • DOI: 10.1007/978-3-642-19440-5_3
  • T1C: Behavioural Semantics
    • The paper focuses on specifying (and evaluating) model semantics for EMF metamodels using reference attribute grammars.
  • T3A: Meta-languages
    • It leverages RAGs as the formalism/metalanguage for semantics specification.
  • T4A: Workbenches
    • It introduces JastEMF as tool support integrating EMF with the JastAdd RAG system.
