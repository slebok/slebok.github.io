SLE 2024 paper:
Method Bundles

Dimi Racordon, Dave Abrahams

  • DOI: 10.1145/3687997.3695633
  • T1A: Design
    • It proposes a new language construct that bundles semantically related method variants under one name.
  • T3F: API
    • The motivation is to improve API ergonomics by consolidating variant-heavy interfaces into coherent bundles.
  • T5F: Performance
    • Bundling unlocks compiler optimizations by making relationships between variants explicit.
