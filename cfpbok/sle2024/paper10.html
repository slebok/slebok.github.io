SLE 2024 paper:
Statically and Dynamically Delayed Sampling for Typed Probabilistic Programming Languages

Gizem Çaylak, Daniel Lundén, Viktor Senderov, David Broman

  • DOI: 10.1145/3687997.3695634
  • T1B: Static Semantics
    • A central issue is reconciling delayed sampling with static typing when random variables change representation/types.
  • T4C: Vertical Transformation
    • It introduces compile-time delayed-sampling optimizations that move inference work from runtime to compilation.
  • T4D: Interpretation
    • It also provides runtime support for dynamically delayed sampling when static optimization is insufficient.
  • T5E: Empirical Evaluation
    • Both static and dynamic approaches are evaluated on realistic probabilistic models.
