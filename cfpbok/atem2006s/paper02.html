ATEM 2006 SI paper:
Ontological Approach for the Semantic Recovery of Traceability Links between Software Artefacts

Yonggang Zhang, René Witte, Juergen Rilling, Volker Haarslev

  • DOI: 10.1049/iet-sen:20070062
  • Extended version of A Unified Ontology-Based Process Model for Software Maintenance and Comprehension (ATEM 2006)
  • T5G: Traceability
    • The paper’s central goal is to recover (re-establish) traceability links between documentation and source code at fine granularity.
  • T3E: Ontologies
    • It relies on formal ontological representations and ontology reasoning to infer explicit and implicit relations among artefacts.
  • T2B: Evolution
    • The motivation and application context are software comprehension and maintenance, where missing links hamper evolution activities.
  • T5E: Empirical Evaluation
    • It reports on applying and evaluating the approach (including an application example) to assess the recovered links and reasoning benefits.
