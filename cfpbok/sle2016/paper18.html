SLE 2016 paper:
Side Effects Take the Blame

Felipe Bañados Schwerter

  • DOI: 10.1145/2997364.2997381
  • T1B: Static Semantics
    • It develops gradual effect systems, i.e., static semantics for tracking side effects with gradual guarantees.
  • T1C: Behavioural Semantics
    • Blame strategies are defined at the semantic level for when runtime checks fail.
  • T4D: Interpretation
    • The framework combines static and dynamic checking, involving runtime enforcement mechanisms.
  • T5D: Formal Methods
    • It formalizes properties/strategies for blame in gradual effect systems, aligning with formal-methods-style reasoning.
