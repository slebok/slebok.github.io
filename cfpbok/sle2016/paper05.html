SLE 2016 paper:
Runtime Support for Rule-Based Access-Control Evaluation through Model-Transformation

Salvador Martínez Perez, Jokin García, Jordi Cabot

  • DOI: 10.1145/2997364.2997375
  • T4D: Interpretation
    • The focus is runtime evaluation of access-control policies using model-based techniques.
  • T4B: Horizontal Transformation
    • Model transformation is used as the operational mechanism for evaluating and updating policy decisions.
  • T5A: Requirements
    • Access-control policies are framed as enforcing security requirements (confidentiality/integrity).
