SLE 2016 paper:
BSML-mbeddr: Integrating Semantically Configurable State-Machine Models in a C Programming Environment

Zhaoyi Luo, Joanne M. Atlee

  • DOI: 10.1145/2997364.2997372
  • T1D: Composition
    • It integrates state-machine DSMLs into a C programming environment (mbeddr), focusing on embedding/composition.
  • T3C: DSLs
    • BSML is a family of domain-specific modeling languages for executable state machines.
  • T1C: Behavioural Semantics
    • The work concerns executable semantics for state-machine models.
  • T4C: Vertical Transformation
    • It connects high-level models to executable C-level artifacts via generation/compilation in the environment.
