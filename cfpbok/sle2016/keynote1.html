SLE 2016 keynote:
Redex: A Language for Lightweight Semantics Engineering

Robert Bruce Findler

  • DOI: 10.1145/2997364.2998391
  • T3A: Meta-languages
    • Redex is a dedicated metalanguage for specifying grammars, type systems, and operational semantics.
  • T1C: Behavioural Semantics
    • It supports operational semantics engineering by executing and exploring semantic models.
  • T1B: Static Semantics
    • It includes support for specifying and checking type systems as part of the language models.
  • T5C: Testing
    • It uses random generation/testing to falsify semantic claims about models.
