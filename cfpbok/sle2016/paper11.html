The IDE Portability Problem and Its Solution in Monto

Sven Keidel, Wulf Pfeiffer, Sebastian Erdweg

  • DOI: 10.1145/2997364.2997368
  • T4A: Workbenches
    • Monto is infrastructure for building reusable language services across IDEs (workbench/IDE tooling).
  • T2D: Deployment
    • The central problem is portability of language tooling across different IDE deployments.
  • T3F: API
    • The solution relies on well-defined interfaces/APIs for composing and reusing language services.
