SLE 2016 paper:
Object-Oriented Design Pattern for DSL Program Monitoring

Zoé Drey, Ciprian Teodorov

  • DOI: 10.1145/2997364.2997373
  • T4D: Interpretation
    • It introduces runtime monitoring support for programs written in DSLs.
  • T3C: DSLs
    • The monitoring approach is positioned in the context of DSL development and execution.
  • T5G: Traceability
    • Monitoring relies on collecting/relating execution information, akin to traceability between DSL constructs and runtime events.
