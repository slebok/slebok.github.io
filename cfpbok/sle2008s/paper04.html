SLE 2008 SI paper:
Engineering of Framework-Specific Modeling Languages

Michał Antkiewicz, Krzysztof Czarnecki, Matthew Stephan

  • DOI: 10.1109/TSE.2009.30
  • T3C: DSLs
    • The paper proposes a method and exemplars for engineering framework-specific modelling languages, a form of domain-specific language tailored to a framework.
  • T3F: API
    • The languages are explicitly centred on framework APIs, capturing API abstractions/rules and modelling application usage of APIs.
  • T4A: Workbenches
    • It discusses language tooling/engineering workflows (including reverse/forward/round-trip engineering) needed to support FSML use cases.
