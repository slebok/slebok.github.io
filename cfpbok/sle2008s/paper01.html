SLE 2008 SI paper:
A Flexible Infrastructure for Multilevel Language Engineering

Colin Atkinson, Matthias Gutheil, Bastian Kennel

  • DOI: 10.1109/TSE.2009.31
  • T4A: Workbenches
    • The paper presents and prototypes a new modelling infrastructure that provides built-in support for multilevel modelling, i.e., workbench/tool support for language engineering.
  • T3A: Meta-languages
    • Its core technical contribution is a metamodeling approach that supports multiple ontological classification levels beyond standard linguistic metamodeling.
  • T1D: Composition
    • It explicitly targets combining general-purpose and domain-specific language perspectives over the same models, which is a form of language integration/coordination.
