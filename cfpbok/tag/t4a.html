Tag T4A: Workbenches

All meta-tools, including language workbenches. Work whose primary contribution is meta-tooling for building and using software languages: language workbenches as the most obvious example, but also all other meta-tools, editor and tool integration frameworks, infrastructures for generating or assembling language environments. While language analysis tools can fall into this category, the analyses of language definitions themselves belong in T5B, and transformation pipelines belong in T4B or T4C.
