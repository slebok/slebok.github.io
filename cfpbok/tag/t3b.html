Topic T3B: Transformation Languages

Work whose object of study is a software language to express transformation itself: design, semantics, implementation of transformation DSLs, such as strategies, rewriting languages, query languages, and their expressiveness or correctness. If the main result is a specific transformation pipeline, we can use T4B/T4C.
