Topic T4D: Interpretation

And runtime. Work about executing languages via interpretation and runtime services: interpreters, virtual machines, runtime libraries, debugging and execution services, runtime architectures for language execution. At the same time, semantic definition techniques without a runtime emphasis fit better in T1C.
