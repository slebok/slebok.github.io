Topic T5C: Testing

Work about testing software language implementations or language-defined systems: test generation for DSLs, model-based and property-based testing of software language artefacts, testing semantics, compilers, interpreters, and testing frameworks specific to software language engineering. Formal verification of language properties belongs to T5D.
