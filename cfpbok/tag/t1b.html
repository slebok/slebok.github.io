Topic T1B: Static Semantics

Static Semantics + Type Systems. Work focused on compile-time meaning: type systems, type inference rules, name resolution, scoping, symbol tables, constraint systems, static analyses for correctness, and static checking infrastructure. If the main contribution resides one linguistic layer higher (e.g., grammar well-formedness), we consider T5B instead.
