SLE 2015 paper:
Using decision rules for solving conflicts in extended feature models

Lina Ochoa, Oscar González-Rojas, Thomas Thüm

  • DOI: 10.1145/2814251.2814263
  • T2C: Variability
    • The paper addresses variability management and conflict resolution in feature models for product lines.
  • T4B: Horizontal Transformation
    • It uses model-based transformation/processing to analyze or resolve feature-model conflicts.
  • T5B: Language Description Analysis
    • Conflict detection/resolution is formulated as analysis over the feature-model language description.
  • T5E: Empirical Evaluation
    • The work typically includes evaluation on feature models/scenarios to demonstrate effectiveness (as implied by decision-making scenarios).
