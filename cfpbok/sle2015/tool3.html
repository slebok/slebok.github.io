SLE 2015 tool/demo paper:
Analysis and Transformation with the Nuthatch Tree-Walking Library

Anya Helene Bagge

  • 🏆 Distinguished Tool Paper Award
  • DOI: 10.1145/2814251.2814264
  • T3F: API
    • Nuthatch is presented as a reusable library/API for tree traversal, analysis, and rewriting.
  • T3B: Transformation Languages
    • Its core capabilities include specifying and executing transformations/rewrites over trees.
  • T5B: Language Description Analysis
    • A primary application is program analysis by traversing and collecting information from trees.
  • T4A: Workbenches
    • As a software library for language processing, it supports building language engineering tooling.
