SLE 2015 tool/demo paper:
XMLText: From XML Schema to Xtext

Patrick Neubauer, Alexander Bergmayr, Tanja Mayerhofer, Javier Troya, Manuel Wimmer

  • DOI: 10.1145/2814251.2814267
  • T4C: Vertical Transformation
    • The main contribution is a generative approach that derives Xtext language artefacts from XML Schema schemata.
  • T3C: DSLs
    • It targets XML-Schema-based DSLs and upgrades them toward modern DSL/editor support.
  • T4A: Workbenches
    • Xtext-based generation directly produces language workbench assets such as parsers and IDE support.
  • T2D: Deployment
    • It aims to make DSL tooling more portable/usable across environments by moving to the Xtext ecosystem.
