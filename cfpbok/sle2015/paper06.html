SLE 2015 paper:
Reference attribute grammar controlled graph rewriting: motivation and overview

Christoff Bürger

  • DOI: 10.1145/2814251.2814257
  • T1B: Static Semantics
    • It is centered on semantic analyses implemented via reference attribute grammars (a static-semantics technique).
  • T3A: Meta-languages
    • Reference attribute grammars act as a meta-language for specifying analyses over structured syntax trees/graphs.
  • T5B: Language Description Analysis
    • The paper discusses analysis of language descriptions/attributed specifications and how they control rewriting.
  • T4B: Horizontal Transformation
    • Graph rewriting is used as a transformation mechanism controlled by the attributed specification.
