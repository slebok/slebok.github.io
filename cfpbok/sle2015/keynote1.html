SLE 2015 keynote:
What’s the value of an end user? Platforms and Research: The case of Pharo and Moose

Stéphane Ducasse

  • URL: https://www.sleconf.org/2015/Papers#keynote
  • T4A: Workbenches
    • The keynote is explicitly about building and evolving research platforms (language + analysis platform) as language-engineering infrastructure.
  • T3D: GPLs
    • Pharo is a general-purpose programming language platform, and language/platform design is a recurring theme.
  • T5B: Language Description Analysis
    • Moose is fundamentally about program/software analysis, and several cited results revolve around analyzing code and histories.
  • T2B: Evolution
    • Topics like dynamic core updates and (especially) automated migration-rule generation are directly about evolution and maintenance over time.
