SLE 2023 paper:
Cross-Level Debugging for Static Analysers

Mats Van Molle, Bram Vandenbogaerde, Coen De Roover

  • DOI: 10.1145/3623476.3623512
  • T5D: Formal Methods
    • The contribution targets debugging of static analysers, which are formal-methods-style tools that compute sound/terminating program properties.
  • T4A: Workbenches
    • The work provides tooling support to debug the analyser implementation by relating analysis-level and program-level executions.
  • T5H: Usability
    • It aims to improve the developer experience of diagnosing and fixing bugs in static analysis tools via better debugging support.
