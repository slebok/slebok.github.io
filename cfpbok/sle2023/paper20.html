SLE 2023 paper:
Practical Runtime Instrumentation of Software Languages: The Case of SciHook

Dorian Leroy, Benoît Combemale, Benoît Lelandais, Marie-Pierre Oudot

  • DOI: 10.1145/3623476.3623531
  • T4D: Interpretation
    • The paper proposes practical runtime instrumentation to enable interoperability between software languages during execution.
  • T1D: Composition
    • Interoperability across languages is treated as an integration problem, bridging otherwise siloed language runtimes.
  • T5G: Traceability
    • Defining a behavioral interface and instrumenting executions provides traceable runtime interactions between languages.
