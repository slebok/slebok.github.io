SLE 2023 paper:
Exceptions all Over the Shop: Modular, Customizable, Language-Independent Exception Handling Layer

Walter Cazzola, Luca Favalli

  • DOI: 10.1145/3623476.3623513
  • T1D: Composition
    • The paper proposes a modular, language-independent exception-handling layer that can be plugged into different language implementations.
  • T1C: Behavioural Semantics
    • Exception handling is defined as a reusable semantic layer that affects program execution behavior at runtime.
  • T2B: Evolution
    • The contribution is motivated and validated through a language evolution scenario extending existing languages with new exception features.
