SLE 2023 paper:
Sharing Trees and Contextual Information: Re-imagining Forwarding in Attribute Grammars

Lucas Kramer, Eric Van Wyk

  • DOI: 10.1145/3623476.3623520
  • T3A: Meta-languages
    • The contribution is a technique within the attribute-grammar meta-language paradigm for structuring and reusing semantic computations.
  • T1B: Static Semantics
    • Forwarding and contextual information sharing primarily concern static analyses such as name resolution and type checking.
  • T1D: Composition
    • The work emphasizes modular feature extensions by translating new features to a core while sharing analysis results.
