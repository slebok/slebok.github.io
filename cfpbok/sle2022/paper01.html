SLE 2022 paper:
A Multi-target, Multi-paradigm DSL Compiler for Algorithmic Graph Processing

Houda Boukham, Guido Wachsmuth, Martijn Dwars, Dalila Chiadmi

  • DOI: 10.1145/3567512.3567513
  • T4C: Vertical Transformation
    • The core contribution is a compiler pipeline that generates code for multiple targets/paradigms from a DSL.
  • T3C: DSLs
    • It is centered on a domain-specific language for algorithmic graph processing.
  • T2D: Deployment
    • Multi-target compilation directly supports portability across execution platforms.
  • T5F: Performance
    • Graph-processing compilation choices are motivated by efficiency of the generated programs.
