SLE 2022 paper:
From Coverage Computation to Fault Localization: A Generic Framework for Domain-Specific Languages

Faezeh Khorram, Erwan Bousse, Antonio Garmendía, Jean-Marie Mottu, Gerson Sunyé, Manuel Wimmer

  • DOI: 10.1145/3567512.3567532
  • T5C: Testing
    • It provides testing techniques (coverage and fault localization) for domain-specific languages.
  • T3C: DSLs
    • The framework is explicitly aimed at DSLs and their bespoke semantics/tooling.
  • T4A: Workbenches
    • It is presented as a generic framework that can be instantiated for different DSLs.
