SLE 2009 paper:
Closing the Gap between Modelling and Java

Florian Heidenreich, Jendrik Johannes, Mirko Seifert, Christian Wende

  • DOI: 10.1007/978-3-642-12107-4_25
  • T4B: Horizontal Transformation
    • It provides bidirectional conversion between Java text and model representations (parser/printer), i.e., mapping/conversion.
  • T3A: Meta-languages
    • It defines a full Java metamodel and related specifications, which is a meta-language/metamodeling contribution.
  • T1B: Static Semantics
    • It reflects Java’s static semantics via cross-references and resolution, emphasizing static-semantics handling.
