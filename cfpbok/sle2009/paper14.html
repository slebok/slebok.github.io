Natural and Flexible Error Recovery for Generated Parsers

  • DOI: 10.1007/978-3-642-12107-4_16
  • T4A: Workbenches
    • It improves generated parser tooling by adding better, more robust error recovery.
  • T5H: Usability
    • Error recovery aims to improve user experience by producing more helpful behavior/messages when parsing fails.
  • T4D: Interpretation
    • Parser behavior at runtime (recovery strategy) is a key part of the execution model of generated parsers.
