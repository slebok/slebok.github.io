SLE 2009 paper:
Study of an API Migration for Two XML APIs

Thiago Tonelli Bartolomei, Krzysztof Czarnecki, Ralf Lämmel, Tijs van der Storm

  • DOI: 10.1007/978-3-642-12107-4_5
  • T3F: API
    • The paper studies migration between two XML APIs, treating APIs as the primary language-like artifacts.
  • T2B: Evolution
    • API migration is a form of software/language evolution and maintenance driven by change over time.
  • T4B: Horizontal Transformation
    • Migration typically requires systematic mapping/conversion between old and new API usages.
