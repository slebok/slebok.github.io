SLE 2009 paper:
Language Boxes: Bending the Host Language with Modular Language Changes

Lukas Renggli, Marcus Denker, Oscar Nierstrasz

  • DOI: 10.1007/978-3-642-12107-4_20
  • T1D: Composition
    • Language boxes provide modular, scoped language extensions to a host language—an integration/modularity contribution.
  • T3C: DSLs
    • They target embedding fine-grained DSL extensions into general-purpose host languages.
  • T4A: Workbenches
    • The approach includes tool integration concerns like syntax highlighting and compilation transformations.
