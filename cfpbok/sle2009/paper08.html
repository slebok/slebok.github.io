SLE 2009 paper:
Model Transformation Languages Relying on Models as ADTs

Jerónimo Irazábal, Clàudia Pons

  • DOI: 10.1007/978-3-642-12107-4_10
  • T3B: Transformation Languages
    • It concerns the design/semantics of model transformation languages, using models as abstract data types.
  • T3A: Meta-languages
    • Treating models as ADTs implicates a metalanguage viewpoint for defining transformation languages.
  • T5D: Formal Methods
    • Using ADTs typically brings formal reasoning about correctness and properties of transformations.
