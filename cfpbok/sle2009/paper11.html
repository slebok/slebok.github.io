SLE 2009 paper:
Integration of Data Validation and User Interface Concerns in a DSL for Web Applications

Danny M. Groenewegen, Eelco Visser

  • DOI: 10.1007/978-3-642-12107-4_13
  • T3C: DSLs
    • The paper presents a DSL that integrates validation and UI concerns for web applications.
  • T1D: Composition
    • Its main idea is integrating multiple concerns (data validation + UI) in one language.
  • T5H: Usability
    • A key motivation is improving the developer/user interaction with forms and validation feedback (usability).
