SLE 2021 keynote:
Live Programming and Programming by Example: Better Together

Sorin Lerner

  • URL: https://conf.researchr.org/details/gpce-2021/gpcegpce+2021+-+20th+international+conference+on+generative+programming%3A+concepts+%26+experiences2021/17/GPCE-Keynote-Live-Programming-and-Programming-by-Example-Better-Together
  • T4A: Workbenches
    • The keynote is about interactive environments that provide continual execution feedback and synthesise code from examples, i.e., advanced programming support tooling.
  • T4G: AI-for-SLE
    • Programming-by-example in this context is fundamentally synthesis/AI-style automation driven by examples, so AI techniques are central to the contribution.
  • T5H: Usability
    • The core motivation is improving the programmer experience via continual feedback and higher-level interaction (examples instead of manual coding).
  • T4D: Interpretation
    • Live programming hinges on runtime execution and continuous observation of values during program execution.
