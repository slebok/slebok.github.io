SLE 2013 paper:
A Language Independent Task Engine for Incremental Name and Type Analysis

Guido Wachsmuth, Gabriël Konat, Vlad Vergu, Danny M. Groenewegen, Eelco Visser

  • DOI: 10.1007/978-3-319-02654-1_15
  • T1B: Static Semantics
    • The primary contribution is incremental name and type analysis—core static-semantics infrastructure for IDE feedback.
  • T4A: Workbenches
    • It provides method/tool support (a task engine) intended to be embedded in IDEs to deliver responsive analysis.
  • T1D: Composition
    • The approach factors analysis into modular tasks with dependencies, emphasizing compositional structuring of analyses across languages.
