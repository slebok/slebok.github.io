SLE 2013 keynote:
Dark Knowledge and Graph Grammars in Automated Software Design

Don S. Batory, Rui Carlos Gonçalves, Bryan Marker, Janet Siegmund

  • DOI: 10.1007/978-3-319-02654-1_1
  • T4C: Vertical Transformation
    • The keynote frames automated software design as stepwise program derivation using (graph-grammar) transformation chains, i.e., refinement/generative engineering.
  • T3B: Transformation Languages
    • Graph-grammar rewrite rules are used as the main transformation formalism for deriving implementations from designs.
  • T1A: Design
    • It explicitly links the approach to language design/engineering, treating “design spaces” as languages with structured sentences.
