SLE 2013 paper:
Circular Higher-Order Reference Attribute Grammars

Emma Söderberg, Görel Hedin

  • DOI: 10.1007/978-3-319-02654-1_17
  • Original shorter version of Declarative rewriting through circular nonterminal attributes (SLE 2014 SI)
  • T3A: Meta-languages
    • The paper advances attribute-grammar technology (higher-order RAGs with circularity/rewrites), i.e., a core meta-language for language implementation.
  • T5B: Language Description Analysis
    • A central concern is well-defined evaluation in the presence of interacting features (circularity and rewrites), i.e., analysis of language-definition behavior.
  • T1D: Composition
    • It studies how declarative rewrites and circular attributes interact, touching on compositional coordination between specification mechanisms.
