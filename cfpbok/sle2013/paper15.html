SLE 2013 paper:
A Generic Framework for Symbolic Execution

Andrei Arusoaie, Dorel Lucanu, Vlad Rusu

  • DOI: 10.1007/978-3-319-02654-1_16
  • Original shorter version of Symbolic execution based on language transformation (SLE 2014 SI)
  • T5D: Formal Methods
    • Symbolic execution is a formal-methods technique, and the paper presents a generic, semantics-based symbolic execution framework.
  • T1C: Behavioural Semantics
    • The framework assumes a formal operational semantics (term rewriting) and systematically extends it to enable symbolic execution.
  • T4C: Vertical Transformation
    • It automatically generates a derived (symbolic) language definition from an original semantics, i.e., a generative refinement of semantics artifacts.
