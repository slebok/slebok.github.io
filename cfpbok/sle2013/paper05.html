SLE 2013 paper:
Software Evolution to Domain-Specific Languages

Stefan Fehrenbach, Sebastian Erdweg, Klaus Ostermann

  • DOI: 10.1007/978-3-319-02654-1_6
  • T2B: Evolution
    • The contribution proposes a process for evolving existing systems toward (embedded) DSL usage rather than rewriting from scratch.
  • T3C: DSLs
    • The target outcome is adoption of embedded domain-specific languages to improve maintainability and tool support.
  • T3B: Transformation Languages
    • The proposed evolution process relies on systematic program changes/refactorings that can be viewed as transformation steps toward DSL usage.
