sle2013/tool2

SLE 2013 tool/demo paper:
Method and Tool Support for Classifying Software Languages with Wikipedia

Ralf Lämmel, Dominik Mosen, Andrei Varanovich

  • DOI: 10.1007/978-3-319-02654-1_14
  • T3E: Ontologies
    • The main contribution is a method/tool for building a classification (taxonomy/ontology-like structure) of software languages using Wikipedia categories.
  • T4F: Visualisation
    • The tool visualises the extracted Wikipedia category graph in a navigable, tree-like form to support exploration and understanding.
  • T5B: Language Description Analysis
    • It performs analysis/mining over a large knowledge base (Wikipedia) to derive and refine language-classification structures.
