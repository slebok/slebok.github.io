SLE 2017 paper:
A Formalisation of Parameterised Reference Attribute Grammars

Scott Buckley, Anthony M. Sloane

  • DOI: 10.1145/3136014.3136024
  • T3A: Meta-languages
    • The paper formalises parameterised reference attribute grammars as a meta-language formalism.
  • T5D: Formal Methods
    • Its main contribution is a formalisation enabling equivalence/correctness-style reasoning about attribute-grammar systems.
  • T1B: Static Semantics
    • Reference attribute grammars are widely used to define static semantic analyses in language implementations.
  • T1D: Composition
    • Parameterisation is aimed at modular reuse/composition of attribute-grammar components across language fragments.
