SLE 2017 paper:
A development environment for the Alf language within the MagicDraw UML tool (tool demo)

Ed Seidewitz

  • DOI: 10.1145/3136014.3136028
  • T4A: Workbenches
    • The paper presents an IDE/development environment (in MagicDraw) for the Alf action language.
  • T1D: Composition
    • It focuses on integrating a textual action language with UML models inside an existing modeling tool.
  • T3C: DSLs
    • Alf is a domain-specific action language for expressing executable behavior in UML models.
  • T6B: Industrial
    • The work is demonstrated in a commercial/industrial-grade modeling tool context (MagicDraw).
