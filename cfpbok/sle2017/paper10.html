SLE 2017 paper:
Energy efficiency across programming languages: how do energy, time, and memory relate?

Rui Pereira, Marco Couto, Francisco Ribeiro, Rui Rua, Jácome Cunha, João Paulo Fernandes, João Saraiva

  • DOI: 10.1145/3136014.3136031
  • T5E: Empirical Evaluation
    • This is an empirical study measuring energy, time, and memory across many programming languages.
  • T3D: GPLs
    • The comparison spans general-purpose programming languages as the primary objects of study.
  • T5F: Performance
    • It analyzes performance-related dimensions (runtime and memory) alongside energy consumption.
  • T6D: Synergies
    • It bridges programming languages with energy/performance measurement methodology, a cross-area synergy.
